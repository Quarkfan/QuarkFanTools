# 测试用例集

用于验证 SKILL 回答的准确性和完整性。每个用例包含问题、期望涉及的关键点、期望回答风格。

---

## 政策类

### T001 — 加盟费用咨询

**问题**：加盟魔介要多少钱？

**期望涉及**：
- 四种方案及价格（299/3980/19800/69800）
- 对应产品价值
- 补货折扣差异

**期望风格**：表格清晰列出，附带投资总额提示（约10万以内）

### T002 — 经验要求

**问题**：我没有美业经验，可以做魔介吗？

**期望涉及**：
- 不强制要求美业经验
- 五大赋能全程帮扶

**期望风格**：一句话肯定回答 + 简要说明帮扶体系

## 流程类

### T003 — 回本流程

**问题**：加盟后多久能回本？怎么回本？

**期望涉及**：
- 案例数据（很多门店2天回本收十几万）
- 回本赋能流程（合伙人方案 → 5天预热 → 2天成交培训 → 线上陪跑）
- 合伙人方案档位（2680/4680/9800元）

**期望风格**：先给结论（2天回本案例），再展开流程

### T004 — 抖音送客流程

**问题**：抖音能给我店送客吗？怎么操作？

**期望涉及**：
- 已为61家店送7000+流量，平均每店100+顾客
- 总部承担运营成本，门店只需接待核销
- 抖音搜索"魔介生活美学馆"

**期望风格**：先给数据结论，再说明门店只需接待即可

## 知识库管理

### T011 — 新增政策

**问题**：新增一条政策：2026年7月起，所有新加盟店赠送价值5000元的开业大礼包

**期望操作**：
- 判断分类 → 写入 `03-加盟方案与产品包.md`
- 格式规范，写入成功提示

### T012 — 搜索抖音相关内容

**问题**：搜索知识库里所有关于抖音的内容

**期望操作**：
- 在 knowledge/ 目录搜索"抖音"关键词
- 列出所有匹配文件和内容摘要

### T013 — 答不上的问题

**问题**：魔介的年营业额是多少？

**期望操作**：
- 本地核心知识和 `knowledge/` 没有答案后，继续使用用户态 lark-cli 搜索飞书
- 读取最相关的飞书资料后再回答
- 只有飞书中也没有答案时，才说明现有知识库和飞书资料暂无记录

### T014 — 搜索并分析飞书 PPT

**问题**：飞书里有没有关于如何开店的 PPT？找到后帮我总结重点。

**期望操作**：
- 使用用户态 lark-cli 搜索 `开店`、`拓店`、`门店` 等关键词，并过滤 slides
- 优先检查可用预览；预览不足时导出 PPTX 到当前会话 workspace
- 使用 PowerPoint Skill 和多模态能力结合页面布局、图片、图表与文字总结
- 回答中说明使用的飞书资料名称，不暴露内部命令或本地路径

### T015 — 分析飞书 Excel 或 Word

**问题**：飞书中找一下加盟门店数据表和招商部流程文档，分析开店准备要点。

**期望操作**：
- 本地 knowledge 信息不足时，继续搜索飞书中的 sheet、docx 或普通 Office 文件
- 能预览时先预览；不可预览或信息不足时导出/下载到当前会话 workspace
- Excel 同时检查数据、公式和图表；Word 同时检查正文、表格、图片和复杂排版
- 使用多个来源去重整合，不只返回文件列表

## 测试执行方法

手动逐条提问，验证回答是否：
1. ✅ 包含所有"期望涉及"的关键点
2. ✅ 符合"期望风格"
3. ✅ 不出现 Q 编号
4. ✅ 不编造知识库之外的内容
5. ✅ 本地知识不足时继续搜索飞书
6. ✅ Office 文件充分使用预览、下载和多模态能力
